## 1. 项目概述

实现一个基于浏览器的macOS桌面投屏和远程控制系统。用户通过网页访问并输入密码后，即可实时观看和控制macOS桌面。

### 核心功能
- 实时桌面投屏（WebRTC 10-15fps @ 可配置分辨率）
- 远程鼠标和键盘操作
- 密码认证
- 多档分辨率配置

## 3. 组件设计

### 3.1 macOS Host端（原生应用）

**技术栈**: Swift + ScreenCaptureKit + WebRTC (Google WebRTC SDK)

**核心模块**:

| 模块 | 职责 | 实现方式 |
|------|------|----------|
| ScreenCaptureEngine | 屏幕捕获 | `SCStream` 捕获桌面帧，配置不同分辨率档位 |
| FrameEncoder | 视频编码 | H.264硬件编码通过 `RTCVideoEncoder` |
| WebRTCConnection | WebRTC连接 | 创建 `RTCPeerConnection`，处理SDP交换 |
| InputHandler | 接收并执行远程输入 | WebSocket接收指令，转CGEvent执行 |
| ConfigUI | 分辨率设置界面 | 原生菜单栏/窗口，提供540p/720p/1080p/1440p选项 |

**分辨率档位配置**:
```
- 低画质: 960x540 (适合移动网络)
- 标准画质: 1280x720 (默认推荐)
- 高清画质: 1920x1080 (高速网络)
- 超清画质: 2560x1440 (局域网/高速连接)
```

### 3.2 信令服务器（Node.js）

**技术栈**: Node.js + Express + Socket.io + WebSocket

**职责**:
- 管理员/观看者WebSocket连接管理
- WebRTC SDP/ICE候选者转发
- 简单的密码认证（bcrypt存储hash）
- 输入指令中继（从浏览器到macOS应用）

**API端点**:
```
POST /api/login    - 密码认证，返回JWT
WS /signal         - WebRTC信令通道
WS /input          - 远程输入指令通道
```

### 3.3 浏览器客户端

**技术栈**: 纯HTML/CSS/JS（无框架）

**页面结构**:
1. **登录页**: 密码输入框，认证后进入主界面
2. **主界面**: 全屏视频显示 + 悬浮控制栏
   - 视频画布：显示WebRTC视频流
   - 控制栏：包含分辨率切换、暂停/恢复、断开连接
   - 状态指示：连接状态、延迟、卡顿提示

**远程输入处理**:
- 鼠标移动/点击 → 相对坐标映射 → 发送指令
- 键盘按键 → 捕获并发送 → macOS端模拟
- 支持常见的修饰键（Cmd, Ctrl, Option, Shift）

## 4. 数据流

### 投屏数据流（上行）

```
ScreenCaptureKit捕获 → 原始帧 → VideoToolbox编码(H.264) 
→ WebRTC RTP打包 → ICE通道 → 浏览器Video元素渲染
```

**帧率策略**:
- 目标帧率: 15fps（macOS端限制，兼顾性能和流畅度）
- 根据网络状况动态调整编码码率

### 远程输入数据流（下行）

```
浏览器捕获事件 → 坐标/键码标准化 → WebSocket发送
→ 信令服务器 → WebSocket转发 → macOS应用接收
→ CGEvent创建 → 系统注入执行
```

## 5. 安全设计

### 认证机制

1. **访问认证**: HTTP POST密码登录，成功返回JWT令牌
2. **WebSocket连接**: 携带JWT进行握手验证
3. **令牌有效期**: 24小时可配置，支持手动吊销

### 连接安全

- WebRTC使用DTLS-SRTP加密媒体通道
- 信令WebSocket使用WSS（TLS）
- 输入指令通道同样通过WSS传输

## 7. 技术选型确认

| 组件 | 选型 | 理由 |
|------|------|------|
| 屏幕捕获 | ScreenCaptureKit | macOS官方API，性能最佳，支持窗口级别捕获 |
| 实时传输 | WebRTC | 低延迟P2P，成熟稳定，浏览器原生支持 |
| 信令协议 | Socket.io + WebSocket | 兼容性好，自动降级机制，开发效率高 |
| macOS开发 | Swift | 原生性能，完整系统API支持 |
| 浏览器端 | 原生JS | 无依赖，加载快，WebRTC支持完善 |
